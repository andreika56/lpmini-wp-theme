# LP Mini — WordPress тема

WordPress тема для лендинга оформления загранпаспортов (Visa Guru). Конвертирована из статического HTML с сохранением оригинальной верстки. Контент редактируется через ACF (Advanced Custom Fields).

## Что в репозитории

| Файл | Что делает |
|------|-----------|
| `style.css` | Заголовок темы для WP (обязательный) |
| `functions.php` | Подключение CSS/JS, ACF хелперы |
| `header.php` | Открывающий HTML, GTM, Roistat |
| `front-page.php` | Главная страница со всеми ACF полями |
| `footer.php` | Закрывающий HTML, JS инициализации (owl, маска, скролл) |
| `acf-export.json` | Готовые ACF поля для импорта в админку |

## Установка — пошагово

### 1. Подготовка папки темы

Создай локально папку `gorizontbeton/` (или любое другое имя — это будет slug темы) и положи туда:

- Все 5 PHP/CSS файлов из этого репозитория (`style.css`, `functions.php`, `header.php`, `front-page.php`, `footer.php`)
- Из оригинального репо `lpmini` скопируй папки **как есть**:
- `css/`
- `js/`
- `img/`
- `font/`
- `feedback/`
- `owlcarousel/`

Структура должна получиться такая:

```
gorizontbeton/
├── style.css
├── functions.php
├── header.php
├── front-page.php
├── footer.php
├── acf-export.json
├── css/
├── js/
├── img/
├── font/
├── feedback/
└── owlcarousel/
```

### 2. Заливка темы на WordPress

**Вариант А (через FTP/файловый менеджер хостинга):**
- Загрузи папку `gorizontbeton/` в `/wp-content/themes/`

**Вариант Б (через админку):**
- Заархивируй папку в `gorizontbeton.zip`
- В WP: **Внешний вид → Темы → Добавить новую → Загрузить тему**
- Выбери ZIP, нажми "Установить"

### 3. Активация темы

WP админка → **Внешний вид → Темы** → найди "LP Mini Visa Guru" → **Активировать**

После активации в админке появится плашка с предложением установить ACF — это нормально.

### 4. Установка ACF (Advanced Custom Fields)

WP админка → **Плагины → Добавить новый** → поиск "Advanced Custom Fields" → **Установить → Активировать**.

Бесплатной версии хватает для всех полей кроме Repeater. Repeater (для блоков "Преимущества" и "Галерея отзывов") есть только в ACF Pro. Если нет Pro — повторяющиеся блоки заполнятся дефолтными значениями из кода и просто не будут редактируемыми.

### 5. Импорт ACF полей

WP админка → **Custom Fields → Tools → Import Field Groups**

Загрузи файл `acf-export.json` → **Import**.

После импорта в меню "Custom Fields" появится группа "Главная страница (LP Mini)" со всеми полями.

### 6. Создание главной страницы

WP админка → **Страницы → Добавить новую** → название "Главная" → **Опубликовать**.

Затем: **Настройки → Чтение** → "На главной странице отображать" → **"Статическая страница"** → выбрать "Главная" → сохранить.

### 7. Заполнение контента

Открой страницу "Главная" в админке → внизу будут все ACF поля:
- Заголовки и подзаголовки
- Картинка героя, логотип, фавикон
- Преимущества (3 блока) — кнопка "+ Добавить"
- Галерея отзывов — кнопка "+ Добавить"
- Тексты гарантии и социального доказательства
- Тексты и кнопки формы
- GTM и Roistat ID

Заполни → **Обновить страницу** → открой сайт — все изменения видны.

## Опционально — замена формы на Contact Form 7

Встроенная форма из папки `feedback/` работает как раньше. Но если хочешь админку для заявок и защиту от спама:

1. Установи плагин **Contact Form 7**
2. Создай форму с теми же полями (телефон + кнопка)
3. В настройках формы CF7 укажи классы:
 ```
 [tel* phone id:phone placeholder "+7(___) ___-__-__"]
 [submit class:button class:feedback "Узнать стоимость"]
 ```
4. Скопируй шорткод формы (например `[contact-form-7 id="123"]`)
5. Вставь его в ACF поле "Шорткод Contact Form 7" на главной странице
6. Сохрани — встроенная форма автоматически заменится на CF7

## Что НЕ ломается

- ✅ Вся оригинальная верстка (классы, теги, структура DOM)
- ✅ CSS из `/css/style.css`
- ✅ Owl Carousel (`.slide2`)
- ✅ Маска телефона
- ✅ Fancybox для просмотра отзывов
- ✅ Плавный скролл по якорям
- ✅ Анимация шапки при скролле
- ✅ GTM и Roistat (ID берутся из ACF)

## Если что-то пошло не так

**Сайт без стилей после активации:**
Проверь, что папки `css/`, `js/`, `img/`, `font/`, `feedback/`, `owlcarousel/` лежат внутри папки темы.

**Не видно ACF полей при редактировании страницы:**
- Проверь что плагин ACF активирован
- Проверь что импорт полей прошёл (Custom Fields → Field Groups → должна быть группа)
- Проверь что в настройках чтения выбрана "Статическая главная страница"

**Карусель отзывов не крутится:**
Проверь что jQuery и `owl.carousel.min.js` загружаются (DevTools → Network).

**Форма не отправляет:**
Если используешь встроенную форму, файл `feedback/feedback.js` должен иметь доступ к PHP-обработчику в папке `feedback/`. На некоторых хостингах PHP в этой папке может быть запрещён — переходи на Contact Form 7.
